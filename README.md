# A Personal Blog Built with Hugo

This is my personal blog, built with Hugo, a static site generator written in Go. The blog contains my thoughts, ideas, and experiences on various topics, including technology, programming, and life.

## Running the Blog

This blog is based on Lowkey Hugo theme.
You can discovered more about this theme, visit https://github.com/nixentric/Lowkey-Hugo-Theme

To run the blog locally on your computer, you'll need to follow these steps:

1. Install Hugo on your computer
2. Clone this repository to your local machine
3. Navigate to the project directory in your terminal
4. Run the command `hugo server -D`
5. Open your web browser and go to `http://localhost:1313`
